# FillBlanks Project

FillBlanks is an AI-based Fill-in-the-Blanks Question Generator built using Natural Language Processing.

## Features
- Generates fill-in-the-blank questions from a paragraph
- Uses NLP to detect important entities
- Provides answer key
- Simple graphical interface using Tkinter

## Technologies Used
- Python
- spaCy (NLP library)
- Tkinter (GUI library)
